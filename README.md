# 🎓 AI-Powered Smart Career Guidance Assistant using Prompt Engineering

## 📌 Project Overview

The AI-Powered Smart Career Guidance Assistant is an intelligent web application developed using Python, Streamlit, and AI APIs. The project uses Prompt Engineering techniques to generate personalized career guidance, learning roadmaps, resume analysis, and career-related assistance for students and job seekers.

This project demonstrates how engineered prompts improve AI-generated responses compared to basic prompts.

---

# 🚀 Features

## ✅ Career Guidance System
- Provides personalized career recommendations
- Suggests suitable career paths
- Generates learning roadmaps
- Gives industry trend insights

## ✅ Prompt Engineering Comparison
- Compares basic prompts and engineered prompts
- Demonstrates response quality improvement using prompt engineering

## ✅ Resume Analyzer
- Upload and analyze resume text files
- Detect missing skills
- Suggest ATS improvements
- Recommend career enhancements

## ✅ AI Career Chatbot
- Answers career-related questions
- Provides learning guidance
- Suggests certifications and career opportunities
